# Chime 게임 룰 정리

## 기본 개요  
Chime은 블록을 배치하여 음악을 만드는 퍼즐 게임입니다. 테트리스와 뮤직 시퀀서를 결합한 형태로, 핵심 루프는 매우 단순합니다.
피스 배치 → 쿼드 형성 → 그리드 커버리지 칠하기 → 음악 생성

## 핵심 메커니즘  
### 피스(Piece)  
12가지 펜토미노(5칸짜리 블록) 중 레벨마다 7개를 사용합니다.
이동, 회전이 가능하고, 그리드 위 어디든 자유롭게 배치할 수 있습니다. (테트리스처럼 위에서 떨어지는 게 아닙니다.)

### 비트라인(Beatline)  
음악 박자에 맞춰 그리드를 수직으로 횡단하는 라인입니다.
비트라인이 배치된 피스에 닿으면 해당 블록을 음표(Chime)로 읽어 소리를 생성합니다.
촘촘하게 모인 블록은 더 긴 음악 프레이즈로 변환됩니다.

### 쿼드(Quad)  
3×3 이상의 빈틈 없는 직사각형 블록 덩어리가 쿼드입니다.
쿼드가 형성된 후 비트라인이 지나가면 쿼드가 셋(Set) → 파쇄(Shatter) 되며 해당 영역이 커버리지로 칠해집니다.
퍼펙트 쿼드: 파쇄 후 프래그먼트가 남지 않는 깔끔한 쿼드. 보너스 점수 + 깨끗한 보드 유지.

### 프래그먼트(Fragment)  
쿼드가 파쇄된 후 남은 블록 조각입니다.
비트라인이 지날 때마다 색이 변하며, 5회 비트라인 통과 후 깜빡이기 시작합니다.
깜빡이는 상태에서 비트라인이 한 번 더 지나면 보드 위의 모든 프래그먼트가 사라지고 콤보 멀티플라이어가 리셋됩니다.
프래그먼트 관리가 고득점의 핵심입니다.  

## 커버리지 & 진행 시스템  

쿼드로 그리드를 칠할수록 노래가 더 많이 들립니다.
화면 하단에서 곡의 진행 섹션을 확인할 수 있습니다.
새로운 섹션에 진입할 때마다 타임 보너스가 부여됩니다.
90% 커버리지 달성 시 레벨 클리어 (이후 100%까지 2회 추가 스윕).
타이머가 0이 되면 현재 진행 중인 쿼드가 자동 파쇄되고, 그로 인해 새 섹션에 도달하면 보너스 타임이 추가되어 게임이 계속됩니다. 그렇지 않으면 게임 오버.

점수 시스템

새로운 커버리지에서만 의미 있는 점수를 획득합니다.
콤보 멀티플라이어: 프래그먼트를 떨어뜨리지 않고 연속으로 쿼드를 만들면 배율이 올라갑니다.
퍼펙트 쿼드 보너스가 있습니다.

게임 모드 (Chime Sharp 기준)
모드특징Practice작은 그리드, 위협 없는 연습 모드Standard원작 방식. 쿼드 → 커버리지 → 타임 보너스로 연장Sharp타이머 없음. 대신 프래그먼트가 떨어지면 라이프 소모. 퍼펙트 쿼드로만 라이프 회복Strike90초 제한. 최대한 빠르게 점수 획득Challenge가장 복잡한 보드 + 제한된 피스셋
전략 팁 (공식 가이드북 기반)

벽 근처가 빌드하기 어려우므로 벽부터 먼저 채우는 게 유리합니다.
레벨별 셰이프셋(7개 피스)의 조합을 파악하고, 어떤 피스끼리 잘 맞는지 전략을 세워야 합니다.
Space/Y 버튼으로 블록을 일시적으로 들어올려 커버리지 갭을 확인할 수 있습니다.
90% 달성 후 추가 2회 스윕만 있으므로, 100%를 노리려면 미리 계획해야 합니다.
