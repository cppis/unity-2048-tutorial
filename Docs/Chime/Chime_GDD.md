# Chime
## Game Design Document

**Mobile Casual Puzzle Game**

Version 1.0 | January 2026

---

# 1. Executive Summary

이 프로토타입은 Chime에서 영감받은 캐주얼 퍼즐 게임입니다.  
스마트폰 세로 모드에 최적화되어 한 손으로 플레이할 수 있으며, 블록을 배치하여 패턴을 만들고 펄스에 맞춰 소거하는 것이 핵심입니다.

### 핵심 차별점

- **턴스템**: Quad별 턴+연쇄
- **한 손 조작**: 세로 화면에서 엄지 하나로 모든 조작 가능
- **작은 그리드**: 16x12 그리드로 모바일 화면에 최적화
- **Chain 모드**: 뿌요뿌요 스타일의 연쇄 소거 시스템

### 게임 정보

| 항목 | 내용 |
|------|------|
| 장르 | Casual Puzzle |
| 플랫폼 | iOS, Android |
| 타겟 유저 | 캐주얼 게이머, 퍼즐 마니아 |
| 세션 시간 | 1-10분 |
| 수익 모델 | Free-to-Play + IAP |

# 2. Core Concept

## 2.1 High Concept

> "손끝으로 그리는 연쇄 퍼즐"

PulseBeat는 블록 퍼즐의 전략적 재미와 연쇄 소거의 쾌감을 결합합니다. 플레이어는 블록을 배치하여 Quad를 만들고, 다양한 펄스 시스템으로 소거하는 것을 경험합니다.

## 2.2 Unique Selling Points

1. **턴 시스템** - Quad별 턴 소거 방식
2. **연쇄 소거** - Chain 모드에서 인접 Quad 연쇄 소거
3. **원핸드 UX** - 언제 어디서나 한 손으로 플레이

## 2.3 Target Audience

| 구분 | 대상 |
|------|------|
| Primary | 25-40세, 캐주얼 모바일 게이머 |
| Secondary | 18-35세, 퍼즐/전략 게임 마니아 |
| Tertiary | 40+, Tetris/2048 경험자 |

# 3. Gameplay Mechanics

## 3.1 기본 규칙

### 그리드 구조  

- **크기**: 12열 x 10행  
- block: grid cell 하나를 block 이라고 한다.  
- **중력 없음**: 블록은 회전 가능하고 지정한 그리드에 배치
- 배차 전 블록은 붉은 색 아웃라인에 내부는 투명하고, 배치 후 지정한 색상으로 그리드에 위치한다.  

### 펜토미노(Pentomino)  
5칸 짜리 블록  

배치되기 전에는 투명하게 표시되고(빨강 레드라인) 배치된 후에는 파랑색으로 표시된가.  

### Quad(Quad)  
- 3×3 이상의 빈틈 없는 직사각형 블록 덩어리  
- 쿼드가 형성된 후 일정 턴이 지나가면 파쇄(Shatter) 되고, 해당 영역이 커버리지로 칠해집니다.  
- 퍼펙트 쿼드: 파쇄 후 프래그먼트가 남지 않는  깔끔한 쿼드. 보너스 점수 + 깨끗한 보드 유지.  


### 3.2 턴 시스템

- 개별 Quad 타이머 + 연쇄 소거  

#### 기본 동작  

1. Quad 완성 시 해당 Quad에 **타이머 시작** (기본 4턴)
2. 블록 배치마다 **모든 대기 Quad 타이머 -1**
3. 타이머 0 도달 시 **해당 Quad 소거**
4. 소거 시 Quad 에 속하지 않은 Fragment 는 그대로 남음   

## 3.4 게임 오버 조건

블록 배치 불가 시 게임 오버

# 4. Controls

## 4.1 기본 조작

| 제스처 | 동작 |
|--------|------|
| 드래그 | 블록 이동 (스냅) |
| 왼 쪽 영역 탭 | 90° 시계방향 회전 |
| 오른 쪽 영역 탭 | 90° 반시계방향 회전 |  
| 더블 탭 or 롱프레스 | 즉시 배치 |

## 4.3 접근성 옵션

- 왼손/오른손 모드
- 회전 버튼 표시
- 색맹 모드
- 햅틱 강도 조절
- **연쇄 힌트** (Chain 모드)

## 6. UI/UX Design

## 6.1 메인 메뉴

```
┌─────────────────────────┐
│       PULSEBEAT         │
│     [ TAP TO PLAY ]     │
├─────────────────────────┤
│  Zen   Endless  Classic │
│  Chain  Challenge Daily │
└─────────────────────────┘
```

## 6.2 Chain Mode UI

```
┌─────────────────────────┐
│ Score: 52,300  🔗 Max:5 │
├─────────────────────────┤
│      [8x6 Grid]         │
│   (Quad에 타이머 숫자)     │
│   (인접 Quad 연결 표시)    │
├─────────────────────────┤
│ 💡 연쇄 힌트: A→B→C       │
│   [Next] [Current]      │
└─────────────────────────┘
```

## 6.3 연쇄 연출

```
[Quad A 소거]
    ↓ 0.3초
"2 CHAIN!" (팝업)
[Quad B 소거]
    ↓ 0.3초
"3 CHAIN!" (팝업 + 셰이크)
[Quad C 소거]
    ↓
"TOTAL: +15,200"
```
