#### Ghost 색상

| 상태 | 색상 | 설명 |
|------|------|------|
| 배치 가능 | `(1, 1, 1, 0.8)` 흰색 | GHOST_COLOR_VALID |
| 배치 불가 | `(1, 0.2, 0.2, 0.8)` 빨간색 | GHOST_COLOR_INVALID |

#### Ghost 동작

```
CreateGhostVisuals()
  ├── GhostOutline 컨테이너 생성 (grid.transform.parent 자식)
  ├── Canvas overrideSorting = true, sortingOrder = 100
  └── 외곽 변에만 GhostEdge Image 생성 (내부 투명)

UpdateGhostVisuals()
  ├── IsPositionOnGrid() → 그리드 범위 내 셀 하나라도 있으면 true
  ├── showGhost = ghostEnabled && isOnGrid
  ├── 블록 셀 숨김/표시 (showGhost에 따라)
  ├── canPlace → 흰색 / 빨간색 아웃라인 색상 변경
  └── 컨테이너 위치 업데이트
```

#### 블록 셀 크기

| 상태 | 크기 | 설명 |
|------|------|------|
| 이동 중 (그리드 위) | `cellSize` | 그리드 라인 노출 |
| 드래그 고스트 | `cellSize` | 보더도 cellSize 기준 |
| 드래그 섀도우 | `cellSize` | 섀도우도 cellSize 기준 |
| 배치 후 | `cellSize` | PlacedBlocks 컨테이너 |

#### 배치 프로세스

```
Place()
  ├── DestroyGhostVisuals()
  ├── 모든 cellObj.SetActive(true) — 고스트에 의해 숨겨진 셀 복원
  ├── placedColor = HSV(h, s*0.6, v)
  ├── for each cell:
  │   ├── grid.SetCellOccupied(pos, true, placedColor, blockId)
  │   ├── cellObj → PlacedBlocks 컨테이너로 이동
  │   ├── cellRect.sizeDelta = (cellSize, cellSize)
  │   ├── cellImage.color = placedColor
  │   └── RemoveBevelChildren() — 잔여 베벨 제거
  ├── 인접 셀 사이 채움 조각(Fill) 생성
  ├── cellObjects.Clear()
  └── Destroy(gameObject)
```

public static void ApplyPalette(QubeBlockShape[] shapes)
{
    if (shapes == null) return;
    for (int i = 0; i < shapes.Length; i++)
    {
        if (shapes[i] == null) continue;  // null 안전
        shapes[i].blockColor = PALETTE[i % PALETTE.Length];
    }
}

```
DetectQuads()
  ├── for each unvisited occupied cell:
  │     ├── FloodFill() → 연결 영역

  ├── 드래그 중: ghostFill (alpha 0.15) + ghostBorder (alpha 0.7)
  │
  └── 배치 시: HSV(h, s×0.6, v) → placedColor
       │
       ├── Quad 아닐 때: placedColor 그대로
       └── Quad일 때: placedColor × 1.3 (밝게)
```

## 4. Ghost 시스템 상세

### 4.1 구조

```
GhostOutline (Canvas overrideSorting, sortingOrder=100)
└── GhostEdge × N (외곽 변 Image만, 내부 비어있음)
```

### 4.2 아웃라인 생성 규칙

각 셀의 4방향(상하좌우)을 검사하여, 인접 셀이 블록에 속하지 않는 변에만 GhostEdge 생성.

### 4.3 표시 조건

| 조건 | Ghost 표시 | 블록 셀 | 아웃라인 색상 |
|------|-----------|--------|-------------|
| 그리드 범위 내 + 배치 가능 | 표시 | 숨김 | 흰색 (0.8 alpha) |
| 그리드 범위 내 + 배치 불가 | 표시 | 숨김 | 빨간색 (0.8 alpha) |
| 그리드 범위 밖 | 숨김 | 숨김 | - |
| ghostEnabled = false | 숨김 | 표시 | - |

### 4.4 생명주기

```
Initialize() → CreateGhostVisuals()
Move() / SetPosition() → UpdateGhostVisuals()
Place() → DestroyGhostVisuals() + 셀 활성화 복원
OnDestroy() → DestroyGhostVisuals()
```

  ├── 그리드 위: SetPosition() → UpdateVisuals() + UpdateGhostVisuals()
