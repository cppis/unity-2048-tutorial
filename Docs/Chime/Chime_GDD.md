# Chime
## Game Design Document

**Puzzle Game**

Last Update: 2026.02.04  

---

## 1. Executive Summary

Chime은 펜토미노 블록을 그리드에 배치하여 Quad를 만드는 전략적인 퍼즐 게임입니다.
일정 턴이 지나면 쿼드가 자동으로 파쇄되며 커버리지를 칠하여 진행합니다.

### 핵심 게임 루프

```
피스 배치 → 쿼드 형성 → n턴 대기 → 쿼드 자동 파쇄 → 커버리지 칠하기
```

### 핵심 차별점

- **턴 기반 파쇄**: 쿼드 생성 후 n턴이 지나면 자동 파쇄
- **타이밍 관리**: 여러 쿼드의 타이머를 전략적으로 관리
- **프래그먼트 관리**: 쿼드 파쇄 후 조각 관리가 고득점의 핵심
- **퍼펙트 쿼드**: 깔끔한 배치로 보너스 획득

### 게임 정보

| 항목 | 내용 |
|------|------|
| 장르 | Puzzle |
| 플랫폼 | PC, Console, Mobile |
| 타겟 유저 | 퍼즐 게이머, 전략 게임 선호자 |
| 세션 시간 | 5-15분 (레벨당) |
| 원작 | Zoë Mode (2010) |

## 2. Core Concept

### 2.1 High Concept

> "전략적 타이밍과 공간 관리의 퍼즐"

Chime은 테트리스의 공간 퍼즐과 타이밍 전략을 결합합니다. 플레이어는 펜토미노 블록을 배치하여 쿼드를 만들고, 각 쿼드의 파쇄 타이밍을 전략적으로 관리합니다.

### 2.2 Unique Selling Points

1. **턴 기반 파쇄** - 쿼드 생성 후 자동 파쇄 타이밍 관리
2. **다중 타이머 관리** - 여러 쿼드의 타이머를 동시에 전략적으로 조절
3. **커버리지 진행** - 시각적 진행도 시스템
4. **프래그먼트 전략** - 정리와 위험의 균형

### 2.3 Target Audience

| 구분 | 대상 |
|------|------|
| Primary | 20-40세, 퍼즐 게임 경험자 |
| Secondary | 전략 게임 팬, Tetris/Lumines 플레이어 |
| Tertiary | 캐주얼 게이머, 타이밍 퍼즐 선호자 |

## 3. Gameplay Mechanics

### 3.1 기본 규칙

#### 그리드 구조

- **크기**: 12×9 (고정)
- **중력 없음**: 블록은 배치된 위치에 고정
- **자유 배치**: 그리드 위 어디든 배치 가능 (테트리스와 다름)

#### 피스 (Pentomino)

**12가지 펜토미노 블록** (각 5칸):

```
F형: ■■    I형: ■     L형: ■      N형: ■      P형: ■■
    ■■          ■         ■          ■■        ■■
     ■          ■         ■          ■          ■
                ■         ■■         ■

T형: ■■■   U형: ■ ■   V형: ■      W형: ■      X형:  ■
     ■         ■■■       ■          ■■        ■■■
     ■                   ■■■        ■■         ■

Y형: ■      Z형: ■■
    ■■           ■
     ■           ■■
     ■
```

- **레벨당 7개 피스** 사용 (셰이프셋)
- **이동 + 회전** 가능
- 겹침 불가, 그리드 밖 배치 불가

#### 피스 시각 상태

**배치 전 (프리뷰)**:
- 아웃라인만 표시 (테두리만 있는 투명한 형태)
- 반투명 상태로 그리드 위에 겹쳐 표시
- 이동/회전 가능
- **배치 가능 여부에 따라 색상 구분**:
  - ✅ **배치 가능**: 흰색 아웃라인
  - ❌ **배치 불가능**: 빨강 아웃라인 (겹침/그리드 밖)

**배치 후 (확정)**:
- 지정된 색상으로 완전히 채워짐
- 그리드에 고정
- 더 이상 이동 불가

```
배치 가능:        배치 불가능:      배치 후:
┌───┬───┐        ┌───┬───┐        ┌───┬───┐
│ ○ │ ○ │        │ ⊗ │ ⊗ │   →   │ ■ │ ■ │
│ ○ │   │        │ ⊗ │   │        │ ■ │   │
└───┴───┘        └───┴───┘        └───┴───┘
(흰색)            (빨강)            (채워짐)
```

#### 배치 불가능 조건

다음 경우 빨강 아웃라인으로 표시:
1. **겹침**: 다른 블록과 겹침
2. **그리드 밖**: 그리드 경계 넘어감
3. **장애물**: Challenge 모드의 장애물과 겹침

### 3.2 턴 시스템

#### 핵심 메커니즘

**피스 배치 = 1턴**

모든 쿼드는 생성된 시점부터 턴을 카운트하며, 일정 턴이 지나면 자동으로 파쇄됩니다.

```
[피스 배치] → [턴 +1] → [모든 대기 쿼드 턴 -1] → [턴 0 도달] → [쿼드 파쇄]
```

#### 작동 방식

1. **턴 증가**: 피스를 그리드에 배치할 때마다 **전역 턴 +1**
2. **쿼드 생성**: 쿼드가 형성되면 **타이머 시작** (기본 4턴)
3. **타이머 감소**: 피스 배치마다 **모든 대기 쿼드의 타이머 -1**
4. **자동 파쇄**: 타이머가 **0에 도달**하면 해당 쿼드 자동 파쇄

```
턴 1: 피스 배치 → Quad A 형성 (타이머 4)
턴 2: 피스 배치 → Quad A (타이머 3) | Quad B 형성 (타이머 4)
턴 3: 피스 배치 → Quad A (타이머 2) | Quad B (타이머 3)
턴 4: 피스 배치 → Quad A (타이머 1) | Quad B (타이머 2)
턴 5: 피스 배치 → Quad A (타이머 0) [파쇄!] | Quad B (타이머 1)
```

### 3.3 쿼드 (Quad)

#### 정의

**3×3 이상의 빈틈 없는 직사각형** 블록 덩어리

```
쿼드 O:              쿼드 X:
┌────┬────┬────┐    ┌────┬────┬────┐
│ ■  │ ■  │ ■  │    │ ■  │    │ ■  │
├────┼────┼────┤    ├────┼────┼────┤
│ ■  │ ■  │ ■  │    │ ■  │ ■  │ ■  │
├────┼────┼────┤    ├────┼────┼────┤
│ ■  │ ■  │ ■  │    │ ■  │ ■  │ ■  │
└────┴────┴────┘    └────┴────┴────┘
   3×3 OK              빈틈 있음 X
```

#### 쿼드 생애주기

1. **형성**: 3×3+ 직사각형 완성 → 쿼드 하이라이트 + 타이머 시작 (4턴)
2. **대기**: 턴마다 타이머 -1 (음표 지속 생성)
3. **경고**: 타이머 1 도달 → 색상 변화 (레드) + 깜빡임
4. **파쇄(Shatter)**: 타이머 0 도달 → 쿼드 자동 파쇄
5. **커버리지**: 쿼드 영역이 **커버리지로 칠해짐** (진행도 증가)

#### 퍼펙트 쿼드 (Perfect Quad)

**파쇄 후 프래그먼트가 남지 않는 깔끔한 쿼드**

```
┌──────────────────┐
│ ■■■■   ■■■      │  ← 3×4 쿼드 (12칸)
│ ■■■■   ■■■      │     +
│ ■■■■   ■■■      │     3×3 쿼드 (9칸)
└──────────────────┘     = 21칸 (퍼펙트!)

파쇄 후:
┌──────────────────┐
│ ░░░░   ░░░       │  ← 프래그먼트 0개
│ ░░░░   ░░░       │     보너스 점수!
│ ░░░░   ░░░       │     깔끔한 보드 유지
└──────────────────┘
```

**보너스**: 점수 배율 + 깨끗한 보드 유지

### 3.4 프래그먼트 (Fragment)

#### 정의

쿼드가 파쇄된 후 **남은 블록 조각**

```
배치:                 쿼드 형성:           파쇄 후:
┌──────┬──────┐      ┌──────┬──────┐     ┌──────┬──────┐
│ ■■■  │      │  →  │[■■■] │      │ →  │ ░░░  │      │
│ ■■■  │ ■    │      │[■■■] │ ■    │     │ ░░░  │ ■    │
│ ■■■  │ ■    │      │[■■■] │ ■    │     │ ░░░  │ ■    │
└──────┴──────┘      └──────┴──────┘     └──────┴──────┘
                      3×3 쿼드             2개 프래그먼트 발생
```

#### 프래그먼트 생애주기

1. **생성**: 쿼드 파쇄 시 쿼드 밖 블록이 프래그먼트로 전환
2. **색상 변화**: 턴마다 색상 변화 (5단계)
   - 1턴: 밝은 색
   - 2턴: 조금 어두워짐
   - 3턴: 더 어두워짐
   - 4턴: 매우 어두움
   - 5턴: 깜빡이기 시작 (경고)
3. **소멸 경고**: 깜빡이는 상태에서 1턴 더 경과
4. **전체 소멸**: 보드 위 **모든 프래그먼트 동시 사라짐**
5. **콤보 리셋**: 콤보 멀티플라이어 초기화

#### 전략적 중요성

**프래그먼트 관리가 고득점의 핵심**:

- ✅ **퍼펙트 쿼드 지향**: 프래그먼트 최소화
- ✅ **프래그먼트 재활용**: 새 피스로 프래그먼트 포함 쿼드 만들기
- ⚠️ **소멸 타이밍 주의**: 5턴 후 깜빡임 시작
- ⚠️ **콤보 유지**: 프래그먼트 떨어뜨리지 않아야 멀티플라이어 유지

### 3.5 커버리지 & 진행 시스템

#### 커버리지 메커니즘

쿼드로 그리드를 칠할수록 **진행도가 증가합니다**.

```
0%        25%       50%       75%       90%      100%
├─────────┼─────────┼─────────┼─────────┼─────────┤
```

#### 진행 섹션

화면 하단에서 진행도 확인:

```
┌────────────────────────────────────────────────┐
│ ░░░░░░░░░░░░░░░░░░░░░░░░■■■■■■■■                │
│ 0%    25%    50%    75%    90%    100%         │
└────────────────────────────────────────────────┘
```

- **새로운 섹션 진입**: 타임 보너스 부여 (Standard 모드)
- **90% 달성**: 레벨 클리어 조건
- **100% 도전**: 90% 이후에도 계속 플레이 가능

#### 레벨 클리어

1. **90% 커버리지 달성** → 레벨 클리어
2. **100% 달성** 시 추가 보너스 점수

### 3.6 타이머 시스템 (Standard 모드)

#### 작동 방식

```
타이머 감소 → 0초 도달 → 현재 쿼드 자동 파쇄 → 새 섹션 도달?
                                                    ↓
                                            Yes: 보너스 타임 추가
                                            No:  게임 오버
```

#### 타임 보너스

| 섹션 진입 | 보너스 타임 |
|----------|-----------|
| 25% | +30초 |
| 50% | +45초 |
| 75% | +60초 |
| 90% | +30초 |

#### 전략

- 섹션 경계 근처에서 큰 쿼드 준비
- 타이머 임박 시 쿼드 형성으로 타임 연장

### 3.7 점수 시스템

#### 기본 원칙

**새로운 커버리지에서만 의미 있는 점수 획득**

```
첫 번째 쿼드:     두 번째 쿼드:
┌────┬────┐      ┌────┬────┐
│ ░░ │    │  →  │ ░░ │ ░░ │
│ ░░ │    │      │ ░░ │ ░░ │
└────┴────┘      └────┴────┘
  +1000점          +1000점 (새 커버리지만)

중복 쿼드:
┌────┬────┐
│ ░░ │    │
│ ░░ │    │
└────┴────┘
  +100점 (소량)
```

#### 콤보 멀티플라이어

프래그먼트를 떨어뜨리지 않고 연속으로 쿼드를 만들면 배율 증가:

| 연속 쿼드 | 배율 |
|----------|------|
| 1 | x1.0 |
| 2 | x1.5 |
| 3 | x2.0 |
| 4 | x2.5 |
| 5+ | x3.0 |

**리셋 조건**: 프래그먼트 소멸 시

#### 보너스 점수

| 조건 | 보너스 |
|------|--------|
| 퍼펙트 쿼드 | +500 |
| 큰 쿼드 (4×4+) | +1000 |
| 90% 달성 | +5000 |
| 100% 달성 | +10000 |
| 콤보 유지 | 배율 적용 |

## 4. Game Modes

### 4.1 모드 요약

| 모드 | 특징 | 위협 | 난이도 |
|------|------|------|--------|
| Practice | 작은 그리드, 위협 없음 | 없음 | ★☆☆☆☆ |
| Standard | 원작 방식, 타이머 | 타임 오버 | ★★★☆☆ |
| Sharp | 라이프 시스템 | 프래그먼트 소멸 | ★★★★☆ |
| Strike | 90초 스피드런 | 타임 오버 | ★★★★★ |
| Challenge | 복잡한 보드 + 제한 피스 | 타임 오버 | ★★★★★ |

### 4.2 Practice Mode

**연습 및 학습 모드**

| 항목 | 내용 |
|------|------|
| 그리드 | 12×9 |
| 타이머 | 없음 |
| 프래그먼트 | 영구 유지 (소멸 안 함) |
| 게임 오버 | 없음 |
| 목표 | 기본 메커니즘 학습 |

#### 특징

- 위협 없는 샌드박스
- 실험 및 연습 가능
- 쿼드 타이머 조절 가능 (4~8턴)
- 튜토리얼 힌트 제공

### 4.3 Standard Mode

**원작 클래식 모드**

| 항목 | 내용 |
|------|------|
| 그리드 | 12×9 |
| 타이머 | 시작 3분 |
| 진행 | 쿼드 → 커버리지 → 타임 보너스 |
| 게임 오버 | 타이머 0초 + 새 섹션 미진입 |
| 목표 | 90% 커버리지 달성 |

#### 핵심 메커니즘

1. 블록 배치 → 턴 증가 → 쿼드 형성
2. 쿼드 타이머 0 도달 → 자동 파쇄 → 커버리지 증가
3. 새 섹션 진입 → 타임 보너스
4. 90% 달성 → 레벨 클리어

#### 전략

- 벽 근처부터 채우기 (벽이 가장 어려움)
- 섹션 경계에서 큰 쿼드 터뜨리기
- 프래그먼트 최소화로 콤보 유지
- 타이밍 관리: 타이머 임박 시 쿼드 준비

### 4.4 Sharp Mode

**라이프 기반 고난이도 모드**

| 항목 | 내용 |
|------|------|
| 그리드 | 12×9 |
| 타이머 | **없음** |
| 라이프 | 3개 |
| 게임 오버 | 라이프 0 |
| 목표 | 90% 커버리지 + 최고 점수 |

#### 라이프 시스템

```
프래그먼트 소멸 → 라이프 -1
퍼펙트 쿼드 → 라이프 +1 (최대 3)
```

- **위협**: 프래그먼트 관리 실패 시 라이프 소모
- **회복**: 퍼펙트 쿼드로만 라이프 회복
- **전략**: 타이머 없지만 라이프 관리 필수

#### 고난이도 요소

- 큰 그리드 (12×9)
- 복잡한 셰이프셋
- 프래그먼트 관리가 생존의 핵심

### 4.5 Strike Mode

**90초 스피드런**

| 항목 | 내용 |
|------|------|
| 그리드 | 12×9 |
| 타이머 | **90초 고정** |
| 연장 | 불가 |
| 게임 오버 | 90초 경과 |
| 목표 | 최대한 빠르게 점수 획득 |

#### 특징

- 타임 보너스 없음
- 순수 스피드 챌린지
- 효율적인 쿼드 배치가 핵심
- 리더보드 경쟁

### 4.6 Challenge Mode

**최고 난이도**

| 항목 | 내용 |
|------|------|
| 그리드 | 복잡한 형태 (L자, ㄷ자 등) |
| 피스 | 제한된 셰이프셋 (난이도 높은 조합) |
| 타이머 | 2분 (짧음) |
| 게임 오버 | 타임 오버 |
| 목표 | 90% 커버리지 달성 |

#### 난이도 요소

- **복잡한 그리드**: 빈 공간, 장애물, 비정형 형태
- **까다로운 피스**: 맞추기 어려운 7개 조합
- **짧은 타이머**: 빠른 판단 필요

## 6. Strategy Tips

### 6.1 공식 가이드북 전략

#### 1. 벽부터 채우기

```
벽 근처가 가장 채우기 어려움
→ 벽부터 먼저 블록 배치
```

**이유**: 중앙은 나중에도 채우기 쉬움

#### 2. 셰이프셋 파악

**레벨별 7개 피스 조합 분석**:

```
L형 + I형 = 3×5 직사각형 가능
T형 + U형 = 4×4 정사각형 가능
```

- 각 레벨의 7개 피스 조합 기억
- 어떤 피스끼리 잘 맞는지 학습

#### 3. 커버리지 갭 확인

**Space / Y 버튼 활용**:

```
[버튼 누름] → 현재 피스 일시 들어올림
           → 아래 커버리지 확인
           → 빈 공간 파악
```

#### 4. 100% 완성 계획

```
90% 달성 → 레벨 클리어
         → 100% 도전 시 남은 영역 계획
```

- 90% 전에 100% 경로 미리 설계
- 남은 영역을 효율적으로 채우기

### 6.2 고급 전략

#### 퍼펙트 쿼드 지향

```
나쁜 예:                    좋은 예:
┌────┬────┬────┐          ┌────┬────┬────┐
│ ■■■│ ■  │    │          │ ■■■│ ■■ │    │
│ ■■■│    │    │          │ ■■■│ ■■ │    │
│ ■■■│    │    │          │ ■■■│    │    │
└────┴────┴────┘          └────┴────┴────┘
  3×3 + 1 프래그먼트          3×3 (퍼펙트!)
```

#### 프래그먼트 재활용

```
1. 프래그먼트 발생
2. 다음 피스로 프래그먼트 포함 쿼드 만들기
3. 프래그먼트 정리 + 점수 획득
```

#### 콤보 유지

- 연속 퍼펙트 쿼드로 멀티플라이어 증가
- 프래그먼트 5턴 전에 정리 (소멸 방지)
