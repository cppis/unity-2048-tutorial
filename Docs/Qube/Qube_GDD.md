# QUBE
## Game Design Document

**Mobile Casual Puzzle Game**

Version 1.5 | January 2026

---

# 1. Executive Summary

PulseBeat는 Chime과 Lumines에서 영감받은 캐주얼 퍼즐 게임입니다.  
스마트폰 세로 모드에 최적화되어 한 손으로 플레이할 수 있으며, 블록을 배치하여 패턴을 만들고 펄스에 맞춰 소거하는 것이 핵심입니다.

### 핵심 차별점

- **트리플 펄스 시스템**: 전역 턴 / 시간 기반 / Quad별 턴+연쇄
- **한 손 조작**: 세로 화면에서 엄지 하나로 모든 조작 가능
- **작은 그리드**: 8x6 그리드로 모바일 화면에 최적화
- **Chain 모드**: 뿌요뿌요 스타일의 연쇄 소거 시스템

### 게임 정보

| 항목 | 내용 |
|------|------|
| 장르 | Casual Puzzle |
| 플랫폼 | iOS, Android |
| 타겟 유저 | 캐주얼 게이머, 퍼즐 마니아 |
| 세션 시간 | 1-10분 |
| 수익 모델 | Free-to-Play + IAP |

# 2. Core Concept

## 2.1 High Concept

> "손끝으로 그리는 연쇄 퍼즐"

PulseBeat는 블록 퍼즐의 전략적 재미와 연쇄 소거의 쾌감을 결합합니다. 플레이어는 블록을 배치하여 Quad를 만들고, 다양한 펄스 시스템으로 소거하는 것을 경험합니다.

## 2.2 Unique Selling Points

1. **트리플 펄스 시스템** - 모드별 다른 소거 방식
2. **연쇄 소거** - Chain 모드에서 인접 Quad 연쇄 소거
3. **원핸드 UX** - 언제 어디서나 한 손으로 플레이
4. **적응형 난이도** - 캐주얼부터 하드코어까지

## 2.3 Target Audience

| 구분 | 대상 |
|------|------|
| Primary | 25-40세, 캐주얼 모바일 게이머 |
| Secondary | 18-35세, 퍼즐/전략 게임 마니아 |
| Tertiary | 40+, Tetris/2048 경험자 |

# 3. Gameplay Mechanics

## 3.1 기본 규칙

### 그리드 구조

- **크기**: 8열 x 6행
- **중력 없음**: 블록은 배치된 위치에 고정
- 빈 셀은 검정색, 채워진 셀은 스킨 색상

### 블록 타입

**기본 블록** (4종):

| 블록 | 모양 | 설명 |
|------|------|------|
| L형 | ■■ / ■ | 3칸, L자 |
| I형 | ■■■ | 3칸, 직선 |
| T형 | ■■■ / _■ | 4칸, T자 |
| O형 | ■■ / ■■ | 4칸, 정사각형 |

**고급 블록** (Chain 모드 고득점 구간):

| 블록 | 모양 | 설명 |
|------|------|------|
| S형 | _■■ / ■■ | 4칸, S자 |
| Z형 | ■■ / _■■ | 4칸, Z자 |

### Quad 시스템

- **Quad**: 2x2 이상의 연속된 채워진 셀 블록
- Quad 완성 시 "대기" 상태로 전환 (하이라이트)
- 펄스 조건 충족 시 소거
- 크기별 점수: 2x2=100, 3x3=300, 4x4=600...

---

## 3.2 펄스 시스템

### 트리플 펄스 시스템

| 방식 | 적용 모드 | 특징 |
|------|----------|------|
| **전역 턴 기반** | Zen | N턴마다 모든 Quad 동시 소거 |
| **Quad별 턴 + 연쇄** | Chain | 개별 타이머 + 인접 연쇄 |

---

### 3.2.1 전역 턴 기반 펄스

**Zen 모드 전용**

1. 블록 배치마다 **전역 턴 +1**
2. N턴 도달 시 **모든 Quad 동시 소거**
3. 턴 카운터 초기화

```
[배치] → [턴 +1] → [4턴 도달] → [펄스!] → [모든 Quad 소거]
```

---

### 3.2.2 Quad별 턴 + 연쇄 (Chain 모드) ⭐

**Chain 모드 전용** - 개별 타이머 + 연쇄 소거

#### 기본 작동

1. Quad 완성 시 해당 Quad에 **타이머 시작** (기본 4턴)
2. 블록 배치마다 **모든 대기 Quad 타이머 -1**
3. 타이머 0 도달 시 **해당 Quad 소거**
4. 소거 시 **인접(4방향) 대기 Quad도 연쇄 소거**

```
[Quad A 완성] → [타이머 4]
[배치] → [타이머 3]
[배치] → [타이머 2]
[배치] → [타이머 1]
[배치] → [타이머 0] → [Quad A 소거!]
                            ↓
                    [인접 Quad B 있음?]
                            ↓ Yes
                    [Quad B 연쇄 소거!] → "2 CHAIN!"
                            ↓
                    [인접 Quad C 있음?]
                            ↓ Yes
                    [Quad C 연쇄 소거!] → "3 CHAIN!"
```

#### 인접 판정

**4방향 인접** (상하좌우, 대각선 제외):

```
      [인접]
        ↑
[인접] ← Quad → [인접]
        ↓
      [인접]
```

**인접 조건**: 두 Quad가 최소 1셀 이상 맞닿으면 인접

```
인접 O:              인접 X:
┌──┬──┬──┬──┐       ┌──┬──┬──┬──┐
│AA│AA│BB│BB│       │AA│AA│  │BB│
│AA│AA│BB│BB│       │AA│AA│  │BB│
└──┴──┴──┴──┘       └──┴──┴──┴──┘
   ↑ 맞닿음            ↑ 떨어짐
```

#### 연쇄 보너스

| 연쇄 | 배율 | 연출 |
|------|------|------|
| 1 (단독) | x1.0 | 기본 소거 |
| 2 | x1.5 | "2 CHAIN!" |
| 3 | x2.0 | "3 CHAIN!" + 셰이크 |
| 4 | x3.0 | "4 CHAIN!" + 강한 셰이크 |
| 5+ | x5.0 | "AMAZING!" + 플래시 |

#### 연쇄 시각 연출

```
[0.0초] Quad A 소거 시작
[0.3초] Quad A 페이드아웃 완료
[0.4초] "2 CHAIN!" 팝업 + Quad B 소거 시작
[0.7초] Quad B 페이드아웃 완료
[0.8초] "3 CHAIN!" 팝업 + 화면 셰이크 + Quad C 소거
...
[연쇄 종료] "TOTAL: +12,500" 보너스 표시
```

# 4. Controls

## 4.1 기본 조작

| 제스처 | 동작 |
|--------|------|
| 드래그 | 블록 이동 (스냅) |
| 탭 | 90° 시계방향 회전 |
| 더블탭 | 90° 반시계방향 회전 |
| 스와이프 다운 | 즉시 배치 |
| 롱프레스 | 확대 모드 |

## 4.2 Undo 시스템

- 마지막 배치 1회 취소 (무료)
- 소거 발생 후 Undo 불가
- **Chain 모드**: Undo 시 타이머 변화도 복구

## 4.3 접근성 옵션

- 왼손/오른손 모드
- 회전 버튼 표시
- 색맹 모드
- 햅틱 강도 조절
- **연쇄 힌트** (Chain 모드)

## 5.2 Zen Mode

| 항목 | 내용 |
|------|------|
| 펄스 | 전역 턴 기반 (4턴마다) |
| 시간 제한 | 없음 |
| 게임 오버 | 없음 |
| 목표 | Coverage 100% |

### 특징
- 수동 펄스 버튼
- 무한 Undo 옵션
- 힌트 시스템

---

## 5.3 Chain Mode  

**연쇄 소거 전략 모드**

| 항목 | 내용 |
|------|------|
| 펄스 | **Quad별 턴 + 연쇄** |
| 시간 제한 | 없음 |
| 게임 오버 | 블록 배치 불가 |
| 해금 | 플레이어 레벨 20 |
| 목표 | 최대 연쇄 + 최고 점수 |

### Quad 타이머

| 난이도 | 타이머 | 블록 |
|--------|--------|------|
| Easy | 5턴 | 기본 4종 |
| Normal | 4턴 | 기본 4종 |
| Hard | 3턴 | +S형, Z형 |

### UI 표시

```
┌─────────────────────────┐
│ Score: 52,300  🔗 Max:5 │  ← 최대 연쇄 기록
├─────────────────────────┤
│   ┌──┬──┬──┬──┬──┬──┐   │
│   │  │AA│AA│BB│BB│  │   │
│   ├──┼──┼──┼──┼──┼──┤   │
│   │  │AA│AA│BB│BB│  │   │  ← A-B 인접
│   ├──┼──┼──┼──┼──┼──┤   │
│   │  │② │CC│CC│④ │  │   │  ← 숫자: 남은 턴
│   ├──┼──┼──┼──┼──┼──┤   │
│   │  │  │CC│CC│  │  │   │
│   └──┴──┴──┴──┴──┴──┘   │
│                         │
│ 💡 A(2)→B(4)→C 연쇄 가능  │  ← 연쇄 힌트
└─────────────────────────┘
```

### 타이머 색상

| 남은 턴 | 색상 |
|--------|------|
| 4-5 | 🟢 그린 |
| 2-3 | 🟡 옐로우 |
| 1 | 🔴 레드 + 깜빡임 |

### 연쇄 힌트 (설정)

- **ON**: 연쇄 가능 경로 표시
- **OFF**: 힌트 없음 (보너스 x1.1)

### 전략 포인트

1. **인접 배치**: Quad를 붙여서 배치
2. **트리거 관리**: 연쇄 시작 Quad 타이머 조절
3. **경로 설계**: 최대 연쇄 경로 미리 계획
4. **타이밍 분리**: 의도치 않은 연쇄 방지

### 리더보드
- Chain 모드 전용 (점수 + 최대 연쇄)
