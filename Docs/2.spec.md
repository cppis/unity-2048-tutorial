#### 메서드

| 메서드 | 시그니처 | 설명 |
|--------|---------|------|
| `Awake` | `void` | 싱글톤 설정 |
| `Start` | `void` | `OnPulse` 이벤트 구독, `NewGame()` 호출 |
| `OnDestroy` | `void` | 이벤트 구독 해제 |
| `NewGame` | `public void` | 점수/상태 초기화, 그리드 클리어, 첫 블록 생성 |
| `SpawnNewBlock` | `private void` | 랜덤 블록 생성 (4종 중), `CanPlaceAnywhere()` 실패 시 `GameOver()` |
| `Update` | `void` | 게임오버 시 R키 재시작, 그 외 `HandleInput()` |
| `HandleInput` | `private void` | WASD/방향키 이동, Q/E 회전, Space 배치 |
| `PlaceBlockCoroutine` | `private IEnumerator` | 배치 → 1프레임 대기 → `IncrementTurn()` → 0.2초 대기 → 새 블록 |
| `OnPulseTriggered` | `private void(int)` | 펄스 점수 누적 |
| `UpdateUI` | `private void` | `"Score: {score}"`, `"Turn: {N} \| Quads: {M}"` |
| `GameOver` | `public void` | 게임오버 텍스트 표시, 현재 블록 제거 |

#### 입력 매핑

| 키 | 동작 | 조건 |
|----|------|------|
| A / LeftArrow | `Move(Vector2Int.left)` | `!isGameOver && currentBlock != null` |
| D / RightArrow | `Move(Vector2Int.right)` | 상동 |
| W / UpArrow | `Move(Vector2Int.up)` | 상동 |
| S / DownArrow | `Move(Vector2Int.down)` | 상동 |
| Q | `Rotate(clockwise: false)` | 상동 |
| E | `Rotate(clockwise: true)` | 상동 |
| Space | `PlaceBlockCoroutine()` | 상동 + `!isPlacingBlock` |
| R | `NewGame()` | `isGameOver` |

#### 블록 배치 시퀀스

```
Space 키 입력
  │
  ├── CanPlace() 실패 → 종료 (로그)
  │
  └── isPlacingBlock = true
       ├── currentBlock.Place()
       ├── currentBlock = null
       ├── yield return null  (1프레임 대기)
       ├── pulseSystem.IncrementTurn()
       ├── UpdateUI()
       ├── yield return WaitForSeconds(0.2f)
       ├── SpawnNewBlock()
       │    └── CanPlaceAnywhere() 실패 → GameOver()
       └── isPlacingBlock = false
```

#### 셀 상태 전이

```
[빈 셀] ──SetOccupied(true)──→ [점유]
  │                               │
  │   SetOccupied(false,          │  SetOccupied(false,
  │   wasCleared=false)           │  wasCleared=true)
  │         │                     │         │
  └────←────┘                     └────→────┘
                                  [소거 흔적]
                                    │
                              clearTimer-- (매 턴)
                                    │
                              clearTimer == 0
                                    │
                                    ▼
                                 [빈 셀]
```

#### 비주얼 상태 매핑

| 상태 | Image.color | Image.enabled | Border | Timer |
|------|------------|--------------|--------|-------|
| 빈 셀 | `#2D3436` | true | 비활성 | 숨김 |
| 점유 (일반) | `transparent (0,0,0,0)` | true | 비활성 | 숨김 |
| 점유 (Quad) | `originalColor × 1.3` | true | 경계만 활성 (노란색) | 중앙 셀만 표시 |
| 소거 흔적 | `#141A1C` | true | 비활성 | 숨김 |

**점유 시 투명 처리 이유**: 배치된 블록의 비주얼(`PlacedBlocks` 컨테이너의 Image)이 셀 위에 별도로 렌더링됨. 셀 Image는 Border/Outline이 작동하려면 enabled 상태를 유지해야 하므로 투명으로 설정.

#### Border 구조

```
셀 (80×80) 기준:

        ┌─ TopBorder (80×6, y=+40) ─┐
        │                            │
  Left  │                            │  Right
 Border │       셀 Image              │ Border
 (6×80  │                            │ (6×80
  x=-40)│                            │  x=+40)
        │                            │
        └─ BottomBorder (80×6, y=-40)┘
```

각 Border는 독립 Image로 Quad 경계만 선택적 활성화 가능.

#### 메서드

| 메서드 | 시그니처 | 설명 |
|--------|---------|------|
| `SetColor` | `void(Color)` | `cellColor` + `image.color` 설정 |

#### Wall Kick 오프셋 순서

```
[0] (0,0)   원래 위치
[1] (-1,0)  왼쪽
[2] (1,0)   오른쪽
[3] (0,1)   위
[4] (0,-1)  아래
[5] (-1,1)  왼쪽 위
[6] (1,1)   오른쪽 위
[7] (-1,-1) 왼쪽 아래
[8] (1,-1)  오른쪽 아래
```

9개 오프셋을 순서대로 시도하여 첫 번째 유효한 위치에서 회전 성공.

| `grid` | `QubeGrid` | `public` | 그리드 참조 |

| `cellObjects` | `List<GameObject>` | `private` | 비주얼 Image 오브젝트 목록 |

#### 스폰 위치 계산

```csharp
position = new Vector2Int(WIDTH / 2 - 1, 1);
// 12x9 그리드: (5, 1)
// 8x6 그리드: (3, 1)
```

#### 비주얼 위치 계산

```csharp
// 그리드 중앙 기준 좌표 변환
float gridWidth  = WIDTH * cellSize + (WIDTH - 1) * spacing;
float gridHeight = HEIGHT * cellSize + (HEIGHT - 1) * spacing;

float leftX   = -gridWidth / 2f;
float bottomY = -gridHeight / 2f;

float xPos = leftX + gridPos.x * cellStep + cellSize / 2f;
float yPos = bottomY + gridPos.y * cellStep + cellSize / 2f;
```

블록은 Canvas 직속 자식이므로 그리드와 동일한 좌표계를 수동 계산.

#### 회전 공식

```csharp
// 시계방향:   (x, y) → (y, -x)
// 반시계방향: (x, y) → (-y, x)
```

#### 배치 프로세스

```
Place()
  ├── placedColor = HSV(h, s*0.6, v)  // 채도 40% 감소
  ├── for each cell:
  │   ├── grid.SetCellOccupied(pos, true, placedColor, blockId)
  │   ├── cellImage.color = placedColor
  │   ├── cellObj.name = "PlacedCell_{x}_{y}"
  │   └── cellObj.SetParent(placedContainer)  // worldPositionStays: true
  ├── cellObjects.Clear()
  └── Destroy(gameObject)  // 블록 오브젝트 제거
```

#### 메서드

| 메서드 | 시그니처 | 설명 |
|--------|---------|------|
| `Initialize` | `void(QubeBlockShape, QubeGrid)` | 초기화, 스폰 위치, 비주얼 생성 |
| `Move` | `void(Vector2Int)` | `CanMove` 체크 후 이동, 비주얼 갱신 |
| `Rotate` | `void(bool)` | Wall Kick으로 회전, 비주얼 재생성 |
| `CanMove` | `bool(Vector2Int)` | 이동 후 위치 유효성 (범위만, 점유 미체크) |
| `CanPlace` | `bool` | 현재 위치에 배치 가능 (범위 + 점유 체크) |

### 2.7 QubeQuadDetector

**파일**: `Assets/Scripts/Qube/QubeQuadDetector.cs`
**역할**: 그리드에서 유효한 직사각형 영역 탐색

#### 알고리즘 흐름

```
DetectQuads()
  │
  ├── for each unvisited occupied cell:
  │     ├── FloodFill() → 연결 영역 (List<Vector2Int>)
  │     └── FindQuadsRespectingBlocks() → 유효 Quad 리스트
  │
  └── return all detected Quads
```

#### FloodFill

- **알고리즘**: BFS (Queue 기반)
- **탐색 방향**: 4방향 (상하좌우)
- **조건**: `IsValidPosition` + `!visited` + `IsCellOccupied`
- **색상 무관**: 점유 여부만 확인

#### FindQuadsRespectingBlocks (Greedy 직사각형 탐색)

```
1. 영역 내 셀을 blockId별 그룹화
2. usedBlocks = {} (사용된 블록 추적)
3. Bounding box 계산 (minX, maxX, minY, maxY)
4. 큰 직사각형부터 작은 순으로 탐색:
   for height = maxH downto 2:
     for width = maxW downto 2:
       for each (startX, startY):
         ├── 직사각형 내 모든 셀이 region에 포함?
         ├── 모든 셀이 occupied?
         ├── 최소 4셀?
         ├── 관련 블록 중 이미 사용된 것 없음?
         └── 관련 블록의 모든 셀이 직사각형에 포함?
              → 유효! Quad 추가, 블록 사용 표시
```

**블록 완전 포함 규칙**: Quad에 포함된 블록은 해당 블록의 모든 셀이 Quad 내에 있어야 함. 블록이 Quad 경계를 넘으면 해당 직사각형은 무효.

#### HighlightQuads

| 단계 | 처리 |
|------|------|
| 1. 전체 리셋 | 모든 점유 셀: `originalColor` 복원, Border 비활성, Timer 숨김 |
| 2. Quad별 하이라이트 | 색상 `× 1.3`, 경계 Border 활성 (노란색), 중앙 셀에 남은 턴 표시 |

**타이머 위치 오프셋**: Quad의 기하학적 중심과 중앙 셀 간 차이를 `cellStep(85)` 단위로 보정.

### 2.8 QubePulseSystem

**파일**: `Assets/Scripts/Qube/QubePulseSystem.cs`
**역할**: 턴 진행, Quad 추적, 소거 관리

#### 상수

| 상수 | 값 | 설명 |
|------|---|------|
| `PULSE_INTERVAL` | `8` | Quad 소거까지 턴 수 |
| `BONUS_MULTIPLIER_TWO` | `1.5f` | 동시 소거 2개 보너스 |
| `BONUS_MULTIPLIER_THREE_PLUS` | `2.0f` | 동시 소거 3개+ 보너스 |
| `REMOVAL_DELAY` | `0.3f` | 소거 후 대기 시간 (초) |

#### 이벤트

```csharp
public delegate void OnPulseDelegate(int score);
public event OnPulseDelegate OnPulse;
```

`RemoveQuads` 코루틴 완료 시 총 점수와 함께 발생.

#### IncrementTurn 처리 순서

```
IncrementTurn()
  │
  ├── isProcessingTurn 중복 호출 방지
  ├── globalTurnCounter++
  │
  ├── Step 0: 모든 셀 UpdateClearTimer() (소거 흔적 페이드)
  ├── Step 1: 기존 Quad turnTimer++ (1턴씩 증가)
  ├── Step 2: DetectQuads() (새 영역 감지)
  ├── Step 3: 각 감지 Quad에 ProcessNewQuad()
  ├── Step 4: turnTimer >= 8 인 Quad → RemoveQuads() 코루틴
  ├── Step 5: HighlightQuads() (시각화 갱신)
  │
  └── isProcessingTurn = false
```

#### ProcessNewQuad 분기 로직

```
ProcessNewQuad(newQuad)
  │
  ├── IsSameQuad() → 기존과 동일하면 무시
  │
  ├── 기존 Quad 순회: overlapping / adjacent 분류
  │
  ├── Case 1: overlapping > 0
  │   └── ProcessOverlappingQuads()
  │       ├── 전체 병합 시도 (overlapping + adjacent)
  │       ├── 성공 & 크기 증가 → 기존 제거, 병합 Quad 추가
  │       └── 실패 → 포함된 Quad 치환 시도
  │
  ├── Case 2: adjacent > 0 (겹침 없음)
  │   └── ProcessAdjacentQuads()
  │       ├── 병합 시도
  │       ├── 성공 & 크기 증가 → 기존 제거, 병합 Quad 추가
  │       └── 실패 → 새 Quad 독립 추가
  │
  └── Case 3: 완전히 새로운 → trackedQuads에 추가
```

#### ProcessOverlappingQuads 상세

```
조건: canMergeAll && mergedQuad.size > totalOldSize
  → 기존 모두 제거, 병합 Quad 추가

조건: !canMergeAll && newQuad.size >= 4
  → 새 Quad에 완전히 포함되는 기존 Quad 찾기
  → 포함된 Quad 제거, 새 Quad 추가 (maxTurnTimer 보존)

그 외: 기존 유지 (로그만)
```

#### ProcessAdjacentQuads 상세

```
조건: canMergeAll && mergedQuad.size > newQuad.size
  → 기존 인접 Quad 제거, 병합 Quad 추가

그 외: 새 Quad 독립 추가
```

#### RemoveQuads 코루틴

```csharp
foreach quad in quadsToRemove:
    totalScore += quad.GetScore()
    foreach cell in quad.cells:
        grid.SetCellOccupied(cell, false, Color.clear, wasCleared: true)
    trackedQuads.Remove(quad)

// 동시 소거 보너스
if count >= 3: totalScore *= 2.0
else if count >= 2: totalScore *= 1.5

OnPulse?.Invoke(totalScore)
yield return WaitForSeconds(0.3f)
```

#### 공개 메서드

| 메서드 | 시그니처 | 설명 |
|--------|---------|------|
| `IncrementTurn` | `void` | 턴 처리 메인 루프 |
| `GetTurnCounter` | `int` | 현재 전역 턴 카운터 |
| `GetPulseInterval` | `int` | 소거 간격 (8) |
| `GetActiveQuads` | `List<QubeQuad>` | 현재 추적 중인 Quad 목록 |
| `ManualPulse` | `void` | 모든 Quad 즉시 소거 |
| `ClearAllQuads` | `void` | 게임 리셋용 (trackedQuads 클리어, 카운터 초기화) |
