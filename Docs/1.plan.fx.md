# Qube 시각 효과 개선 계획

> 난이도 낮음 + 효과 높음 순으로 정렬

## Phase 1: 즉시 적용 가능 (난이도 낮음 / 효과 높음~중간) ✅ 구현 완료

### 1-1. 블록 베벨 효과 ✅
- **구현**: QubeBevel.cs (유틸리티) — 상단/좌측 흰색 하이라이트 + 하단/우측 검정 섀도우
- **적용**: QubeBlock.cs (드래그 중 + 배치 후), QubeBlockPreviewUI.cs (미니셀)
- [x] QubeBevel.cs 신규 작성 (정적 유틸리티)
- [x] QubeBlock.CreateCellVisual()에 베벨 추가
- [x] QubeBlock.Place()에서 배치 크기에 맞게 베벨 재생성
- [x] QubeBlockPreviewUI 미니셀에 축소 베벨 (1.5px) 추가

### 1-2. 색상 팔레트 확장 ✅
- **구현**: QubeBlockShape.cs에 7색 팔레트 상수 + ApplyPalette() 메서드
- **적용**: QubeGameManager.Start()에서 런타임 적용
- [x] 7색 팔레트: 시안, 마젠타, 라임, 앰버, 퍼플, 코랄, 틸
- [x] blockShapes 배열에 순서대로 자동 적용

### 1-3. 배경 그라데이션 + 비네트 ✅
- **구현**: QubeBackground.cs (신규) — 프로시저럴 Texture2D 생성
- **적용**: QubeGameManager.Start()에서 Canvas에 자동 적용
- [x] 수직 그라데이션 (밝은 네이비 → 어두운 네이비)
- [x] 비네트 오버레이 (중심→가장자리 어두워짐)

### 1-4. UI 패널 스타일링 ✅
- **구현**: QubeBlockPreviewUI.cs + QubeScoreFeedback.cs
- [x] 프리뷰 슬롯 라운드 코너 (프로시저럴 스프라이트)
- [x] 첫 번째 슬롯 시안 글로우 보더
- [x] 콤보 텍스트 색상 새 팔레트 반영 (앰버/마젠타)
- [x] 텍스트 글로우 효과 (TMP 언더레이)
- **참고**: 커스텀 폰트는 Unity Editor에서 SDF 에셋 생성 필요 (코드만으로 불가)
